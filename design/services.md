# Services Page Layout Design

## Overview
A comprehensive listing page showcasing all veterinary services offered by the clinic.

**URL:** `/services`

---

## Page Structure

### 1. Header / Navigation Bar
**Position:** Fixed top
**Components:** Same as home page
- "Services" menu item highlighted

---

### 2. Page Hero Section
**Layout:** Full-width banner

**Content:**
- **Headline:** "Our Services"
- **Subheadline:** "Comprehensive veterinary care for your beloved pets"

**Style Notes:**
- Light blue gradient background
- Smaller than homepage hero
- Clean, professional look

---

### 3. Services Grid Section
**Layout:** Grid layout (3 columns on desktop, 2 on tablet, 1 on mobile)

**Title:** "Complete Care for Your Pets"

**Service Cards (6 services):**

1. **General Health Checkups**
   - Icon: Stethoscope
   - Description: Comprehensive wellness exams
   - Link to: `/services/general-checkup`

2. **Inpatient & Emergency Care**
   - Icon: Hospital
   - Description: 24/7 emergency services
   - Link to: `/services/emergency-care`

3. **Vaccination & Prevention**
   - Icon: Syringe
   - Description: Core and non-core vaccines
   - Link to: `/services/vaccination`

4. **Grooming & Hygiene**
   - Icon: Scissors
   - Description: Professional grooming services
   - Link to: `/services/grooming`

5. **Dental Care**
   - Icon: Tooth
   - Description: Complete dental services
   - Link to: `/services/dental-care`

6. **Surgery & Procedures**
   - Icon: Activity
   - Description: Advanced surgical care
   - Link to: `/services/surgery`

**Card Components:**
- Image or icon
- Service title
- Short description
- "Learn More" button

**Style Notes:**
- Consistent card styling
- Hover effects
- Equal height cards

---

### 4. Call-to-Action Section
**Layout:** Full-width banner

**Content:**
- **Headline:** "Ready to Book an Appointment?"
- **Subtext:** "Our team is ready to provide the best care for your pet"
- **CTA Button:** "Book Now" (links to `/booking`)

**Style Notes:**
- Bold, contrasting colors
- Positioned before footer

---

### 5. Footer
**Layout:** Multi-column layout
**Content:** Same as home page

---

## Color Scheme
- **Primary Color:** Calming blue or green
- **Service Cards:** White with shadows
- **CTA:** Brand orange/coral

## Typography
- **Page Title (H1):** 36-48px, bold
- **Service Titles (H3):** 20-24px, semi-bold
- **Body Text:** 16px

## Responsive Design Notes
- Desktop: 3-column grid
- Tablet: 2-column grid
- Mobile: 1-column stack
- Touch-friendly interactions
