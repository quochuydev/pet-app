Installation

Prerequisites: You need Node.js (version 18 or higher) installed on your computer. Download from nodejs.org

Step 1: Extract Files

Extract the downloaded ZIP file to your desired location on your computer.

Step 2: Install pnpm (Package Manager)

This project uses pnpm as the package manager. Open your terminal/command prompt and install it globally:

npm install -g pnpm

Step 3: Install Dependencies

Navigate to the project folder in your terminal and install all required packages:

cd pet-app
pnpm install

This will install all necessary dependencies including Next.js, React, Tailwind CSS, and other libraries.

Step 4: Set Up Database (Optional)

If you want to use the booking system with database functionality:

  1. Create a .env.local file in the root directory
  2. Add your database connection string: 
    DATABASE_URL="your-postgres-connection-string"
JWT_SECRET="your-secret-key-here"
  3. Run database migrations: 
    pnpm db:push
Note: The template works perfectly without a database for static display. The database is only needed if you want the booking system to save appointments.

Step 5: Start Development Server

Start the development server to preview your site:

pnpm dev

Open your browser and visit http://localhost:3000 to see your template in action!

Step 6: Build for Production

When you're ready to deploy your website:

pnpm build
pnpm start
Success! Your template is now installed and running. You can start customizing it to match your pet clinic's branding.

Customization Guide

This guide will help you customize the template to match your pet clinic's branding and needs.

1. Changing Colors

To change the color scheme, edit app/globals.css:

@layer base {
  :root {
    /* Primary color (blue/teal) */
    --primary: 210 100% 50%;

    /* Secondary color (orange/coral) */
    --secondary: 25 95% 58%;

    /* Accent color */
    --accent: 160 84% 39%;

    /* Background colors */
    --background: 0 0% 100%;
    --foreground: 222 47% 11%;
  }
}
Tip: Colors use HSL format. Use a tool like HSL Color Picker to find your desired colors.

2. Changing Logo and Images

Update Logo:

  1. Replace your logo file in the public/ folder
  2. Update the logo reference in app/components/Header.tsx: 
    <div className="text-2xl font-bold text-primary">
  YourClinicName
</div>

Update Images:

  1. Add your images to the public/ folder
  2. Update image paths in components: 
    // In Hero.tsx, Services.tsx, Team.tsx, etc.
<Image
  src="/your-image.jpg"
  alt="Your description"
  width={600}
  height={400}
/>

3. Editing Content

Content for each section can be edited in the respective component files:

Section File to Edit
Navigation Menu app/components/Header.tsx
Hero Section app/components/Hero.tsx
Services app/components/Services.tsx
Team Members app/components/Team.tsx
Why Choose Us app/components/WhyChooseUs.tsx
Footer app/components/Footer.tsx

4. Changing Fonts

The template uses Geist fonts by default. To change fonts:

  1. Import your desired font in app/layout.tsx: 
    import { Inter } from 'next/font/google'

const inter = Inter({ subsets: ['latin'] })
  2. Apply the font: 
    <body className={inter.className}>

5. Adding/Removing Sections

To add or remove sections from the homepage, edit app/page.tsx:

export default function Home() {
  return (
    <>
      <Header />
      <Hero />
      <Services />
      <Team />
      <WhyChooseUs />
      {/* Add your new section here */}
      <CTA />
      <Footer />
    </>
  )
}

6. Customizing the Booking Form

Edit the booking form in app/booking/page.tsx:

  • Add/remove form fields
  • Change validation rules
  • Customize success messages
  • Add email notifications

7. Modifying Tailwind Configuration

For advanced styling customization, edit tailwind.config.ts:

export default {
  theme: {
    extend: {
      colors: {
        primary: 'hsl(var(--primary))',
        secondary: 'hsl(var(--secondary))',
      },
      fontFamily: {
        sans: ['var(--font-geist-sans)'],
      },
      // Add your custom styles here
    },
  },
}
